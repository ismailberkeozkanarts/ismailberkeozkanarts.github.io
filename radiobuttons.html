Bootstrap 5 Radio Butonları

Bu sayfada Bootstrap 5 ile radio butonları yapısının ne olduğunu ve iki farklı görünümünü size göstereceğim.

Radio butonları Nedir?
Radio butonları, çoğunlukla sayfa içerisinde bir veya birden fazla tercih arasından seçim veya seçimler yapmamızı sağlayan, bu yapılan seçim veya seçimlere göre işlem yaptıran butonlardır.

Aşağıda ki kod yapısı ile size basit bir örnek sunmak istiyorum;
Bootstrap 5'in hazır css yapılarındaki class'ları kullanarak form-check class'lı divi oluşturdum. Daha sonra input type için radio özelliğini tanımlayıp, gerekli hazır class'ları vererek ve label kullanarak iki adet radio butonu oluşturdum.Burada input seçim yapacağımız butonu oluştururken, label ise seçim yapacağımız tercihin ismini oluşturur.

<div class="form" style="background-color:darkgreen; color:white;"> <div class="form-check"> <input class="form-check-input" type="radio" name="flexRadioDefault" id="flexRadioDefault1"> <label class="form-check-label" for="flexRadioDefault1"> Ankara </label> </div> <div class="form-check"> <input class="form-check-input" type="radio" name="flexRadioDefault" id="flexRadioDefault2" checked> <label class="form-check-label" for="flexRadioDefault2"> Adana </label> </div> </div>

Inline Radio

Inline radio butonlarının tek farkı aynı satırda yan yana hizalanmalarıdır

Aşağıda ki kod yapısındaki fark ise div içerisinde tanımlı class'ta form-check dışında form-check-inline class'ının ekli olması. Bu sayede radio butonlarımız yan yana aynı hizada yer alacaklardır.

<div class="form" style="background-color:darkgreen; color:white;"> <div class="form-check form-check-inline"> <input class="form-check-input" type="radio" name="inlineRadioOptions" id="inlineRadio1" value="option1"> <label class="form-check-label" for="inlineRadio1">İstanbul</label> </div> <div class="form-check form-check-inline"> <input class="form-check-input" type="radio" name="inlineRadioOptions" id="inlineRadio2" value="option2"> <label class="form-check-label" for="inlineRadio2">İzmir</label> </div> </div>
